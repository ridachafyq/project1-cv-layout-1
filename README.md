# Project 1 – CV – Layout 1

A **responsive CV/Resume template** (Layout 1) with **light & dark themes**, a **theme toggle**, and **print-friendly design**.  

## Features

- Two-column responsive layout (sidebar + main content)  
- Automatic and manual dark mode (saved in `localStorage`)  
- Accessible HTML with ARIA labels and decorative SVGs marked `aria-hidden`  
- Print-optimized with backgrounds removed and sections kept intact
